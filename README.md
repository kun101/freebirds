# Penguin Party - Serverless MMO

This is a browser-based MMO that runs entirely on the client-side using **Firebase Realtime Database**.

## How to Play

1.  **Just Open the App**:
    The app is pre-configured with a default public Firebase database. You do not need to set up anything.
    
2.  **Join with Friends**:
    Since everyone uses the same pre-configured database, just share the URL of this web app with your friends. When they log in, they will appear in the same world as you!

## Features
- Realtime Multiplayer Movement
- Room Switching
- Live Chat
- Avatar Customization
- Serverless Architecture (No backend server required)

## Development

```bash
npm install
npm start
```
